# 🚲 Bikeshare Data Analysis

### 📅 Date created
April 15, 2026

### 📌 Description
This project analyzes bikeshare data from different US cities using Python, providing insights into travel patterns through interactive user input and statistical analysis.
Users can filter the dataset by **city**, **month**, and **day of the week**, allowing for customized exploration of trends and behaviors.

### ✨ Features
- 🎯 **Interactive Filtering**  
    Select city, month, and day to analyze specific subsets of data  
- 📊 **Statistical Insights**  
  - Most common travel times  
  - Popular stations and routes  
  - Trip duration analysis  
  - User demographics  
- 🔍 **Raw Data Exploration**  
  View data in chunks of 5 rows upon request  

### 🛠️ Requirements
Make sure you have the following installed:
- Python 3.x
- pandas
- numpy 

### 📂 Files used
- `new_york_city.csv`
- `chicago.csv`
- `washington.csv`
